PPE Detection & Safety Monitoring System

Real-time construction safety monitoring using object detection, tracking, and rule-based violation detection.

Problem

Construction sites require continuous monitoring to ensure workers wear mandatory personal protective equipment (PPE)—such as helmets and high-visibility vests—to reduce injury risk. Manual monitoring does not scale and leaves gaps in coverage. This system automates safety compliance by detecting PPE and flagging violations in real time from camera feeds.

System Overview

The pipeline ingests a camera stream and runs detection, tracking, and rule-based violation logic before logging events for monitoring.

Camera Stream
→ YOLO detection
→ Multi-object tracking
→ PPE classification
→ Violation detection logic
→ Event logging
